How to run GUI
===========================
## Install:
1. Node [https://nodejs.org/]
2. Grunt [https://www.npmjs.com/package/grunt]

# Production
How to run on production

## Build
To build application:
```
grunt build
```

## Run
To run application:
```
cd distribution
npm start
```

Application is running on port **8081**


# Development
How to develop

## Build & Run

```
npm install
grunt develop
```


Application is running on port **8081**
